# Dino AI Playground

一个基于浏览器的 Google Dino 变体，支持两种自动化训练模式：遗传算法（GA）和强化学习（RL，REINFORCE）。本目录为单页应用，直接用浏览器打开 `index.html` 即可运行，无需构建或依赖安装。

## 快速上手
- 打开 `dino/index.html`，点击「开始进化」或「开始学习」启动对应模式。
- 右上角「GA → RL / RL → GA」切换训练模式，切换时会重置环境并保留当前滑动条数值映射。
- 「手动游戏」可体验手动操作；`Space` 跳跃，`P` 暂停，`Esc` 重置，`R` 仅在手动模式下复活。

## 主要文件
- `js/game.js`：核心游戏循环、渲染与物理逻辑（无算法细节）。
- `js/training.js`：GA/RL 的统一接口（BaseTrainer），屏蔽算法细节对游戏循环的影响。
- `js/geneticAlgorithm.js`：GA 逻辑与谱系跟踪。
- `js/reinforcementLearning.js`：REINFORCE 策略梯度实现，含轨迹存储与梯度更新。
- `js/dino.js`：恐龙实体、特征提取、动作执行与碰撞检测。
- `js/obstacle.js`：障碍物实体与生成器。
- `js/main.js`：UI 交互、模式配置、参数滑动条映射与可视化。
- `GA_BUG_DIAGNOSIS.md`：GA 常见问题与修复记录。
- `MODULES.md`：模块与主要函数索引（维护入口）。

## 参数滑动条映射
同一组滑动条在不同模式下含义不同，`main.js` 的 `MODE_CONFIG` 统一管理：

| 滑动条 | GA 含义 | RL 含义 |
| --- | --- | --- |
| 种群大小 | 种群数量 | Agent 数量（并行采样） |
| 变异率 | 突变概率 | 学习率（值 × 0.2） |
| 交叉率 | 交叉概率 | 奖励缩放系数（值 × 0.2） |
| 精英保留 | 保留个体数 | 折扣因子 γ（0.9 + (值-1)×0.01） |

## 本次改动摘录
- 统一 GA/RL 模式文案与滑动条映射，减少 UI 分支重复。
- `startSimulation` 现在在进入 AI 模式前重置场景，避免手动模式遗留的障碍影响训练。
- 统计计算增加空数组保护，初始状态不再出现 `NaN` 或 `-Infinity`。
- 滑动条范围与 trainer 参数自动同步，避免显示数值与实际参数不一致。
- RL 默认参数更稳定（学习率/奖励系数/并行数量），并增加熵正则与优势裁剪。

## 调试提示
- GA：观察右下角谱系树与进化进度条；停滞时可提高变异率或降低精英数。
- RL：开启 `reinforcementLearning.js` 的 `debug` 标志可打印权重变化与动作分布。
